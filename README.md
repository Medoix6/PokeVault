# PokeVault

[![Vue.js](https://img.shields.io/badge/Vue.js-3-42b883?logo=vue.js&logoColor=white)](https://vuejs.org/)  
[![Pinia](https://img.shields.io/badge/State-Pinia-ffd859?logo=vue.js&logoColor=black)](https://pinia.vuejs.org/)  
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)  
[![Node.js](https://img.shields.io/badge/Node.js-%3E=16-339933?logo=node.js&logoColor=white)](https://nodejs.org/)

A modern **Vue.js** application to browse, search, and edit Pokémon details — powered by **Pinia** for state management.

---

## ✨ Features

- 🔍 Browse and search through a list of Pokémon
- 📖 View detailed information for each Pokémon
- 📝 Edit Pokémon details (name, description, height, weight)
- 💾 State managed and persisted with **Pinia + localStorage**
